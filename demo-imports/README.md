# Demo-Imports

Dieses Verzeichnis enthält Testdaten für den Sectionscode-Import.

- `3-produkte-sammeldatei.txt` ist eine kombinierte Referenzdatei mit drei Beispielprodukten.
- `sectionscode-testpaket/` ist direkt für den aktuellen Importer gedacht.

Im aktuellen Importer gilt:

- eine TXT-Datei = ein Produkt
- Bilder mit gleichem Sectionscode werden automatisch zugeordnet

Beispiele:

- `1.1.txt` + `1.1.png` + `1.1.1.png` + `1.1.2.png`
- `1.2.txt` + `1.2.png` + `1.2.1.png` + `1.2.2.png`
- `1.3.txt` + `1.3.png` + `1.3.1.png` + `1.3.2.png`
