# Quality rules

Use this reference for every run.

## Hard blockers

Do not rate the listing as A-level if any of these remain unresolved:

- product type is unclear
- brand, model, or sold variant is contradictory
- core technical data is missing or conflicts across sources
- unsupported performance, safety, or certification claims remain
- variant grouping is broken
- the image set does not show the actual sold item clearly enough

## Score rubric

Score out of 100 using these buckets:

- Completeness: 0-25
- Consistency across text and images: 0-20
- Buyer clarity and usefulness: 0-15
- SEO and discoverability: 0-15
- Marketplace fit: 0-15
- Variant integrity: 0-10

## Grade mapping

- A: 90-100 and no hard blockers
- B: 80-89 with only minor gaps
- C: 65-79 with visible improvement work still needed
- D: below 65 or major unresolved blockers

## A-level standard

An A-level listing should satisfy all of the following:

- title identifies the product immediately
- bullets answer the main buyer questions fast
- description adds context instead of repeating the bullets
- technical data is structured, consistent, and unit-safe
- search terms widen discovery without keyword spam
- image order is intentional and supports trust
- missing data is either resolved or explicitly called out
- variant logic is clean and understandable

## QA routine

1. Compare title, bullets, description, and attributes against the fact table.
2. Compare the chosen sold variant against the image set.
3. Remove unsupported claims, duplicated wording, and empty promo language.
4. Check whether the most important buyer questions are answered quickly.
5. Confirm that the export block can be mapped without hidden ambiguity.
6. List the exact blockers preventing a higher grade.
